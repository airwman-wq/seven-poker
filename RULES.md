# 7포커 규칙 명세 (이 프로젝트의 기준)

한국 온라인 보드게임(한게임·피망 계열)에서 통용되는 7포커 규칙을 기준으로 한다.
변형이 존재하는 부분은 아래에 채택안을 명시한다.

## 진행

1. 전원 기본금(앤티, 기본 베팅 단위 1장)을 판돈에 낸다.
2. 각자 4장을 받는다 (전부 비공개).
3. 각자 동시에 **1장을 버리고**, 남은 3장 중 **1장을 공개**한다. → 비공개 2 + 공개 1
4. 1차 베팅 (공개 카드가 가장 높은 사람 = 보스부터 시계방향).
5. 4번째 카드(공개) 배분 → 2차 베팅.
6. 5번째 카드(공개) 배분 → 3차 베팅.
7. 6번째 카드(공개) 배분 → 4차 베팅.
8. 7번째 카드(**비공개**) 배분 → 마지막(5차) 베팅. 베팅은 총 5회.
   최종 7장 = 비공개 3장(처음 2 + 마지막 1) + 공개 4장.
9. 쇼다운: 남은 사람들이 7장을 공개, 그중 5장으로 만든 최고 족보가 판돈 전부를 가져간다.

베팅 도중 한 명 빼고 전부 다이하면 즉시 그 사람이 판돈을 가져간다 (패 비공개).

## 베팅 액션

| 액션 | 의미 | 가능 조건 |
|---|---|---|
| 체크 | 추가 베팅 없이 넘김 | 이번 라운드에 아직 아무도 베팅 안 했을 때 |
| 삥 | 기본 단위 1장 베팅 | 이번 라운드 첫 베팅일 때 |
| 콜 | 직전 베팅 금액 맞추기 | 앞에 베팅이 있을 때 |
| 따당 | 직전 베팅액의 2배로 레이즈 | 앞에 베팅이 있을 때 |
| 하프 | 현재 판돈의 절반만큼 레이즈 | 항상 (첫 베팅으로도 가능) |
| 다이 | 포기 | 항상 |

- 칩이 콜 금액보다 적으면 가진 칩 전부로 콜(올인)한다. **사이드팟은 만들지 않는다** (캐주얼 게임머니 단순화 — 올인한 사람이 이기면 판돈 전부 가져감).
- 올인한 사람은 이후 베팅 라운드를 건너뛰고 쇼다운에 참가한다.

## 족보 (높은 순)

| # | 족보 | 설명 |
|---|---|---|
| 13 | 로열 스트레이트 플러시 | 10-J-Q-K-A 같은 무늬 |
| 12 | 백 스트레이트 플러시 | A-2-3-4-5 같은 무늬 |
| 11 | 스트레이트 플러시 | 연속 5장 같은 무늬 |
| 10 | 포카드 | 같은 숫자 4장 |
| 9 | 풀하우스 | 트리플 + 원페어 |
| 8 | 플러시 | 같은 무늬 5장 |
| 7 | 마운틴 | 10-J-Q-K-A (무늬 무관) |
| 6 | 백 스트레이트 | A-2-3-4-5 (무늬 무관) |
| 5 | 스트레이트 | 연속 5장 |
| 4 | 트리플 | 같은 숫자 3장 |
| 3 | 투페어 | 페어 2개 |
| 2 | 원페어 | 페어 1개 |
| 1 | 탑 (하이카드) | 위 어느 것도 아님 |

- 한국식: 마운틴 > 백 스트레이트 > 일반 스트레이트. 둘 다 플러시면 로열 > 백SF > 일반SF.
- 숫자 순위: 2 < 3 < … < 10 < J < Q < K < A.
- **무승부 없음**: 족보·숫자가 완전히 같으면 결정 카드(페어면 페어, 그 외는 최고 카드)의 무늬로 가린다.
- 무늬 순위: ♠ 스페이드 > ◆ 다이아 > ♥ 하트 > ♣ 클로버.

## 보스(베팅 선) 결정

매 베팅 라운드, **공개 카드만으로** 만든 조합이 가장 높은 사람이 먼저 베팅한다.
(포카드 > 트리플 > 투페어 > 원페어 > 하이카드, 같으면 숫자 → 무늬 순.)

## 칩

- 시작 칩 100,000 / 기본 베팅 단위(삥·앤티) 1,000.
- 칩이 0이 되면 다음 판 시작 시 100,000 재충전 (게임머니).
